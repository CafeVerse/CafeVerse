# Security Policy

## Supported Versions

Only the latest version of CafeVerse is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by creating a new draft security advisory under the Security tab of the repository on GitHub:
https://github.com/cafeverse/CafeVerse/security/advisories/new
